# **Sistema-Alesli-Wolf-Tank-Socios**

Sistema integral de gestión y marketing para la florería Aleslí, desarrollado bajo metodología ágil Kanban. Automatiza procesos contables, gestión de clientes (CRM), pedidos con nota de entrega, inventarios de materiales de decoración, centro de datos para floristas, marketing digital, atención al cliente con WhatsApp y chat automatizado.



**Proyecto: Gestión Comercial - Florería Alesli**
